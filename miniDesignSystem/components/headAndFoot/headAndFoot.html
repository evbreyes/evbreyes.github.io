Headers and Footers

This page introduces the navigation headers and footers of my portfolio website on all pages for different screen sizes.

Headers

Desktop / Web Navigation Header

Mobile Navigation Header

Usage & Best Practices

  • The header must be used on all pages to ensure visibility of top level navigation around the whole website.
  • Use a different color on the active top level navigation while making sure the colors have enough contrast for readability.
  • The header must contain a form or variation of the portfolio logo.

Accessibility

  • Must have enough contrast for text and background to ensure easy readability.
  • Use 
    aria
    labels in the navigation code, for example, 
    aria-current="page"
    for the active page.

HTML Example

                
    <header class="sticky-top">
      <nav class="navbar navbar-expand-lg navbar-light">
        <div class="container-fluid">
          <a class="navbar-brand" href="../../index.html">
            <img src="../../images/logos/headingLogo.png" alt="Elmar Reyes Logo" id="logo">
          </a>
          <button class="navbar-toggler" type="button"  data-bs-toggle="collapse" data-bs-target="#navbarNavDropdown" 
          aria-controls="navbarNavDropdown" aria-expanded="false" aria-label="Toggle navigation">
            <span class="navbar-toggler-icon"></span>
          </button>
          <div class="collapse navbar-collapse" id="navbarNavDropdown">
            <ul class="navbar-nav">
              <li class="nav-item">
                <a class="nav-link active" aria-current="page" href="#">Home</a>
                <!-- Projects -->
                <li class="navbar-item">
                  <a class="nav-link" href="#">Projects</a>
                </li>
              <!-- About Me -->
              <li class="navbar-item">
                <a class="nav-link" href="#">About Me</a>
              </li>
              <!-- Contact Me -->
              <li class="navbar-item">
                <a class="nav-link" href="#">Contact Me</a>
              </li> 
            </ul>
          </div>>
        </div>
      </nav>
    </header>

Usage & Best Practices

  • The footer is used on all pages of the website to provide contact information for the portfolio.
  • The footer must contain a form or variation of the portfolio logo.
  • Ensure the links provided are active and connected to active accounts.

Accessibility

  • Must have enough contrast for text and background to ensure easy readability.
